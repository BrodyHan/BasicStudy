# 도서관 미션
> 도서관 책을 완성하는 미션

## 기능 요구 사항
1. [level1](../../level1/library)과 [level2](../../level1/library)의 내용을 기본으로 한다.
2. 기본 메뉴는 `회원 관리`, `책 관리`, `대여 관리`이 존재한다.
3. 책과 회원은 대출 중이거나 대출한 책이 존재할 경우 삭제가 불가능하다.
5. `자바`or `코틀린`의 경우 `IllegalArugmentException`을 발생시켜야한다.
5. 잘못된 입력이 들어오게 되면 `[ERROR]`로 시작하는 에러 메세지  출력 후 다시 받는다.

## 입출력 요구 사항

### 입력
1. 기본 메뉴
    ```
    1
    ```
2. 회원 관리 메뉴, 책 관리 메뉴, 대여 관리 메뉴
    ```
    1
    ```
3. 회원 추가
    ```
    홍길동 01011112222
    ```
4. 회원 삭제
    ```
    홍길동 01011112222
    ```
5. 책 추가
    ```
    Java의 정석, 남궁석, 도우출판, 5
    ```
6. 책 삭제
    ```
    Java의 정석, 남궁석
    ```
7. 대여를 위한 회원 정보
    ```
    홍길동 01011112222
    ```
8. 대여할 책의 정보
    ```
    Java의 정석, 남궁석

    ```

### 츌력
- 도서관 출력 문구
    ```
    도서관 프로그램 입니다.  
    ```
- 기본 메뉴 출력
    ```
    # 메인 메뉴
    1. 회원 관리
    2. 책 관리
    3. 대여 관리
    ```
- 회원 메뉴 출력
    ```
    # 회원 메뉴
    1. 회원 추가
    2. 회원 삭제
    ```
- 회원 추가 및 삭제
    ```
    회원 추가를 성공했습니다.
    회원 삭제를 성공했습니다.
    ```
- 회원 관련 예외
    ```
    해당 회원은 이미 존재합니다.
    ```
  
- 책 관리
    ```
    # 도서 메뉴
    1. 도서 추가
    2. 도서 삭제
    ```
- 도서 추가 및 삭제
    ```
    도서 추가를 성공했습니다.
    도서 삭제를 성공했습니다.
    ```
- 도서 관련 예외
    ```
    해당 책은 이미 존재합니다.
    ```

- 대여 관리
    ```
    # 대여 메뉴
    1. 대여 현황 조회
    2. 대여 하기
    ```
- 대여 현황 조회
    ```
    조회할 회원의 이름과 번호를 입력하세요
    ```
- 대여 하기
    ```
    대여할 회원의 이름과 번호를 입력하세요
    대여할 책의 이름과 지은이를 입력해 주세요.
    ```
